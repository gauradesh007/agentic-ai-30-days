# Agentic AI — 30 Day Learning Journey

## Building Agent Systems from First Principles

This repository documents a structured 30-day journey to learn Agentic AI by building systems from scratch.

The goal was not simply to use AI frameworks, but to understand how modern agent systems work internally through:

- Tool Execution
- Controllers
- Memory Systems
- Planning Systems
- Retrieval Systems
- Reflection Systems
- Self-Correcting Workflows
- Multi-Agent Architectures
- Vector Databases
- RAG
- Human-in-the-Loop Systems
- Production Agent Workflows

Every day introduced one new capability that built on top of the previous day.
